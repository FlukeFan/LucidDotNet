To update the Demo project to the latest (unreleased) version of Lucid:

1.  Build Lucid form the command line to create the 'packaged' folder;
2.  Open the Demo.sln;
3.  Tools->Options->NuGet Package Manager->Package Sources, add a source Lucid.local with the dir (checkout-dir)\packaged;
4.  Open the Package Manager Console, change the 'Package source' to Lucid.local, and run each of the following commands:

Update-Package Lucid.Persistence
Update-Package Lucid.Persistence.Testing
Update-Package Lucid.Persistence.NHibernate
Update-Package Lucid.Facade
Update-Package Lucid.Facade.Testing
Update-Package Lucid.Web
Update-Package Lucid.Web.Testing
